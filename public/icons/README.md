# PWA Icons

This directory contains PWA icons for the application.

## Required Icons

- `icon-192x192.png` - 192x192 pixels (required)
- `icon-512x512.png` - 512x512 pixels (required)
- `icon-144x144.png` - 144x144 pixels (recommended)
- `icon-384x384.png` - 384x384 pixels (recommended)

## Icon Generation

Icons should be created with:
- PNG format
- Transparent or solid background
- Accessible design (meets WCAG requirements)
- Represents the application brand/identity

## Temporary Placeholder

For development, placeholder icons will be generated. Replace with proper branded icons before production.
